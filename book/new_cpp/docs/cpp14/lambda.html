lambda

表达式捕获

在14之前的lambda无法对表达式进行捕获。 

#include <iostream>

int main(void)
{
    auto add = [v1 = 1, v2 = 2 + 2]() -> int 
    {
        return v1 + v2;
    };

    std::cout << add() << std::endl;

    return 0;
}

泛型lambda

lambda函数的形参可以使用auto关键字产生意义上的泛型

#include <iostream>

int main(void)
{
    auto add = [ ](auto x, auto y)
    {
        return x + y;
    };

    std::cout << add(2, 3) << std::endl;

    return 0;
}
