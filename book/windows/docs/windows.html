快捷键

说明：win是windows键

窗口

  • alt + tab
    切换窗口
  • win + tab
    激活任务视图
  • win + d
    显示桌面
  • win + m
    最小化所有窗口
  • win + shift + m
    还原最小化的窗口
  • win + 左/右/上/下
    管理窗口贴合屏幕四个边缘或全屏或小化等

多桌面

  • win + + d
    创建新的虚拟桌面
  • win + + f4
    关闭当前虚拟桌面
  • win + + 左/右
    左右切换虚拟桌面

其他

  • win + e
    开启资源管理器(即我的电脑(文件夹))
  • win + l
    锁定电脑
  • win + u
    系统设置(显示, 声音, 通知, 电源等)
  • win + r
    打开"运行"
  • win + a
    打开通知中心
  • win + t
    后台程序切换
  • win + p
    投影切换
  • win + shift + s
    截图
