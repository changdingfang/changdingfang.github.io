taskset

用于进程cpu调优，可以把某个进程指定在某个cpu内工作。在某些情况下能够提高cpu的整体性能

语法

            <-------- 参数 ----->
task (选项) (cpu-list) (pid | cmd)

选项

  • -a
    对给定pid的所有任务的设置或检索cpu相关性
  • -p
    指定进程号
  • -c
    指定cpu编号(cpu的第几个核心数)

参数

  • cpu编号列表
  • 进程号
  • 命令

使用示例

cpu核心数查看

$ cat /proc/cpuinfo/ | grep "cpu cores" | uniq

查看给定进程号的cpu相关性

$ taskset -p ${pid}

使给定的进程号在1和2核心上工作 

$ taskset -c 1,2 -p ${pid}
