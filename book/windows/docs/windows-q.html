登录界面

问题
win10,输入完密码后在欢迎界面无穷转圈,进不去

解决
进入安全模式中进行修复。

  • 开机时长按住CTRL键，直到出现修复界面，依次点高级选项，疑难解答，高级选项，启动设置，重启之后按F4进入安全模式
  • 用鼠标右键点开始菜单，打开命令提示符（管理员）
  • 输入 netsh winsock reset catalog 提示成功后重启操作系统

github无法显示图片

  1. 获取github可用ip地址
    通过站长工具查看域名绑定的ip
    或者通过命令行使用 ping github.com 来获取可用ip
  2. 添加host映射
    修改hosts文件( C:\Windows\System32\drivers\etc\hosts ), 添加映射
