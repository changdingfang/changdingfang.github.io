wc

用于计算(字符数，行数等)

语法

wc (选项) (参数)

选项

  • -c
    只显示Bytes数
  • -l
    只显示列数
  • -w
    只显示字数

参数

  • 文件
    需要统计的文件列表

使用示例

$ wc -l test.txt
