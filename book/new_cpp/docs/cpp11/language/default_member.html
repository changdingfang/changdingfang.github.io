委托和成员默认初始值

一个构造函数可以调用该类中的其它构造函数来完成部分初始化任务(委托).
声明成员时可以直接指定默认初始值.

#include <iostream>

class A
{
public:
    A(int new_number) : m_nNumber(new_number) { std::cout << "hello world" << std::endl;    }
    A() : A(42) {   std::cout << "hello world 2" << std::endl;  }

private:
    int m_nNumber;
    int m_nValue = 10;

};

int main(void)
{
    A a;
    return 0;
}
