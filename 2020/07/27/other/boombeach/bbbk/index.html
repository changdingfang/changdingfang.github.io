    百科介绍

    维基百科: 《海岛奇兵》（英语：Boomin ranta）是由芬兰游戏厂家Supercell开发的适用于iOS和Android的免费大型多人在线即时战略游戏。它于2013年11月11日在加拿大推出，并于2014年3月26日广泛发布。
    百度百科: 《海岛奇兵》是由芬兰Supercell Oy公司开发，Supercell Oy（国际版）和昆仑游戏（中国大陆地区）发行的一款战斗策略类、全球同服的手机游戏，于2014年8月在中国发布。

    扎卡定理

    当妹子的射程攻击范围有目标的时候，妹子不是无脑地从近到远攻击目标，而是会动脑子的。妹子选择目标的优先权如下

    1. 防御建筑 > 升级中的防御建筑 > 无害建筑
      此条无法判断优先次序的进入下一条
    2. 残血(百分比)少的建筑 > 残血多的建筑 > 无损建筑(=残血量100%)
      此条无法判断优先次序的进入下一条
    3. 距离近的 > 距离远的

    获取资源计算

    注: 这个可能不准确或者根本就不对

    a = 显示的总资源
    v = 保险库存储的资源
    p = 生产建筑存储的资源
    x = 保险库保护百分比
    r = 系统奖励资源
    r2 = 减少系数
    s = 雕像加成百分比
    t = 部落加成百分比

    可获得资源 = (( a - v + p ) * (1 - x) + r ) * ( 1 - r2 ) * ( 1 + s + t )

    当前钻石消耗 = (剩余时间 / 总时间) * 总钻石消耗

    特殊活动

    螃蟹

    简介

    作为恐怖博士最邪恶的超级武器, 超级螃蟹是一只专为破坏而生的甲壳巨兽! 这是对每位司令的终极考验, 也是赢取丰厚资源奖励的绝佳机会.
    超级螃蟹具有的阶段近乎无限, 而且难度逐级递增. 玩家可以在此相互比拼, 夺取排行榜上的一席之地.
    每月最后一个周末都会有超级螃蟹活动, 且每次的超级螃蟹活动都是独一无二的, 提供专属的阶段加成和奖励. 每次超级螃蟹活动开始前, 您都可以查看它的详细信息.

    • 第一次活动于当地时间2016年1月15日下午6点开始，历时7天。
    • 第二次活动于2016年3月25日开始，历时3天。
    • 第三次活动开始于2016年7月1日，历时3天。
    • 第四次活动于2016年10月28日开始，历时4天。
    • 第五次活动于2017年1月27日开始，历时5天4个小时。
    • 第六届于2017年5月26日开始，历时3天。
    • 在2018年3月更新之后，超级螃蟹活动每月一次。

    进攻

    仅当玩家还有剩余攻击时，才能攻击巨型螃蟹。
    在超级螃蟹事件开始时，玩家拥有40次攻击次数。
    在任何阶段结束时都会奖励一次免费攻击。
    在阶段之间, 地板可能会不一样, 这使每次的战斗战场形状都可能不一样.
    阶段可以包含所有常规防御建筑物和所有原型防御。
    每个阶段通常都具有比上一个阶段更强大的防御建筑物和蓝雕像加成。
    想要赢取胜利, 必须摧毁能量核心

    独有的建筑

    • 机器小怪容器
      当机器小怪容器破裂时, 它会释放9个友好的机器小怪, 无论玩家自己的小怪等级如何, 它们都会攻击敌方建筑物.
      机器小怪容器的一个特殊变体用于万圣节附近的大型螃蟹活动. 当破碎时, 它会释放9个类似蝙蝠的友好类蝙蝠生物, 但它们不会触发任何类型的地雷(但是, 它们仍可能受到所述地雷爆炸的影响).
      每个小怪容器的生命值均为270.

    • 冷冻炸弹
      当冷冻炸弹的生命值降低至零并延迟2秒后触发它时，它会释放出大面积冷冻爆炸，冻结其周围的部队和建筑物并引爆地雷。
      冻结效果持续20秒，使攻击和移动速度降低75％。
      每个冷冻炸弹的生命值为270，爆炸造成260的伤害。

    • 加速血清容器
      当加速血清因其生命值降低至零而被触发时，经过短暂的延迟后，它会释放出粉红色的爆炸粉末，从而提高周围所有部队和建筑物的速度。与冷冻炸弹不同，它不会触发地雷。
      每个加速血清有500生命值。

    • 可怕的迫击炮
      可怕的迫击炮是在万圣节前夕的巨型螃蟹活动中出现的一种特殊防御手段。
      它造成的伤害很小，但会释放出南瓜形的外壳，使部队和小怪在受到冲击时感到困惑，使他们不稳定地徘徊了几秒钟。受影响的部队将不会受引导作用。冲击时伴随着险恶的笑声。
      每个可怕的迫击炮具有6,000点生命，每秒造成5点伤害。

    • 博士人偶
      博士人偶站在一个装有紫色扬声器的木盒子上，占据2x2的空间。
      当部队在其范围内时，将播放T博士嘲讽的声音，并吸引附近的部队。
      每位博士人偶都拥有1,400点生命，每秒造成1点伤害。摧毁后，会在小半径内造成400点伤害。

    • 超级士兵大桶
      超级士兵大桶指的是多种不同类型的大桶，它们是根据其内部的部队命名的。
      每个大桶都包含特定部队的非常强大的版本，它将为您而战。
      桶里面的士兵不同，因此名称也不同。他们甚至在生命值上也有所不同。
      他们不会跟随引导弹信号，不会受任何雕像加成影响, 并且不能被医疗. 如果被销毁, 它们也不计入人员伤亡.
      这些大桶内的部队包括：

      • 从步兵大桶中释放出来的超级步兵，生命值为10,000。
      • 从重机枪手大桶释放出超级重机枪手，生命值20,000。
      • 从土著勇士大桶中释放出来超级土著勇士，生命值20,000。
      • 从医生大桶中释放的一名超级医生，生命值15,000。
      • 从投弹兵大桶中释放出超级投弹兵, 生命值15,000。
      • 从机器蝙蝠大桶中释放出超级蝙蝠, 生命值20000。

      一些部队具有被动技能，通常玩家无法使用。
      步兵的射速大大提高，类似于机枪。
      超级土著勇士冻结建筑物的方式和极冻先锋相似。
      超级医师还能加快部队的速度.
      超级投弹兵的榴弹每投掷五枚手榴弹，会造成一次大爆炸。

    商人

    简介

    商人是一个神秘的人物，他每周都乘坐一艘红色的大潜艇访问你的岛屿。
    她在格林尼治标准时间周四午夜到达，周日晚上的同一时间离开。在某些情况下，可能或提前到达后待的更久.

    商人券

    商人券有三种获取方式

    • 收取每日奖励
    • 收取补给箱
    • 从商人的额外物品(如果有的话)中购买.
    • 战斗母舰赛季奖励(获得数量取决于您在战斗母舰的段位)

    前两个选项可以获得一张商人券.
    商人券用于在商人那里购买商人宝箱.

    商人宝箱

    商人宝箱可使用商人券购买。每周第一个商人宝箱是免费的，但接着的每个宝箱将花费您3张商人券。
    宝箱包含随机奖励，包括基本资源，水晶和原型模块，但保证至少可以得到一个英雄券。
    通过这种方式获得的资源可以超过存储容量而不会丢失。

    其他物品

    每次商人到达时，除了提供箱子外，她还提供三种不同的额外物品，来交易其他类型的资源。
    每周有三项额外物品。其中一些额外优惠只能购买一次，而其他优惠则可以多次购买。
    这些交易包括但不限于:

    • 商人券
    • 资源
    • 英雄券
    • 一种特定类型的资源换区另一种特定类型的资源(例如: 木材换金币)
    • 水晶
    • 一种或多种某种类型的水晶，兑换另一种类型的水晶（例如3个绿水晶换3个紫水晶）。

    战斗母舰

    基础简介

    • 当您的司令部达到10级后, 您将获得一艘强大的战斗母舰, 它将助你保卫海岛世界! 有了它, 您可以参加PVP(玩家对玩家)对战!
    • 在战斗母舰对战中获胜可获得荣誉之星. 摧毁8个引擎室后您便可以开启战斗母舰宝箱. 其中包含升级币和解锁币, 您可以用它们来升级您的战斗母舰.
    • 您需要合理使用这些资源升级和解锁防御建筑或兵种, 然后带领您的部队夺取对战胜利, 登上排行榜前列. 升级或解锁科技树上的项目后, 您可以立即使用该项目.
    • 战斗母舰上有一种名为引擎室的特殊建筑, 它们是您为获取胜利而攻击的主要目标. 摧毁所有引擎室, 战斗就会结束.
    • 在一个赛季中积极对战, 达到一定条件后您就可以获取更多的引擎室. 记得把它们放在最万无一失的地方.
    • 每个战斗母舰赛季都有时间限制, 具体结束时间会在游戏内显示.
    • 新赛季开始时, 您的战斗母舰会被重置. 上个赛季结束时您所在的星阶越高, 新赛季开始时您所在的初始星阶也就越高.
      每个赛季的科技树也会有所不同. 不断冲击更高星阶, 并在全新的荣誉之路上赢取更多奖励吧!

    引擎室

    引擎室是战斗母舰的动力枢纽, 保证战斗母舰正常运转. 玩家可通过解锁科技树上的扩建项目来获得更多引擎室.

    科技树

    • 在科技树里, 您可以选择解锁和升级各种项目, 包括防御建筑, 部队, 英雄, 战舰能力, 各种加成等.
    • 战斗母舰宝箱中可以获得解锁币和升级币. 而总司令您需要决定如何使用这些资源!
    • 每个赛季科技树的布局都会发生改变, 这样每个赛季及对战所使用的战术都会有所不同.
    • 您解锁和升级的项目越多, 科技等级就会越高. 达到更高科技等级不仅能增强您战斗母舰的实力, 还能解锁科技树上更多项目.

    可解锁项目

    建筑/登陆艇

    • 这些项目包含各种防御建筑.
    • 每次升级这些项目, 对应建筑的等级都将提升多级, 同时还可以在战斗母舰建造更多该建筑.
    • 相同建筑的等级也相同.

    部队

    • 这些项目包括各种兵种.
    • 每次升级这些项目, 对应兵种的等级都将提升多级, 同时还可以增加对应兵种的储备规模.
    • 储备规模将决定您能在登陆艇上装载某个兵种的最大数量.

    英雄

    • 这些项目包括现有的所有英雄.
    • 每次升级这些项目, 对应的英雄等级和技能等级都将提升多级.

    战舰/战舰能力

    • 这些项目包括战舰和战舰能力.
    • 每次升级这些项目, 对应的战舰或战舰能力都将提升多级.

    引擎室

    • 引擎室项目是特殊的项目, 不仅能增加您战斗母舰上的引擎室数量, 同时还将提升引擎室的等级和扩大您战斗母舰的甲板.
    • 只有当您的主基地司令部达到特定等级, 才能解锁这些引擎室项目(舰长的钥匙可以解决司令部等级限制).
    • 解锁引擎室项目还要求战斗母舰达到特定科技等级. 解锁和升级其他项目可以提高战斗母舰的科技等级.

    三选一项目

    • 三选一项目上会显示一个叹号, 而非某个建筑的图标.
    • 这些项目会给您三个选项, 您可以从中任选一个.
    • 三个项目需要的解锁币相同.
    • 三个项目提供相同数量的科技点数.
    • 每个选项都包含一定数量的某种防御建筑或地雷.
    • 如果该种防御建筑或地雷还没有在之前的项目中解锁, 那您就无法选择该选项.
    • 通过三选一项目解锁的建筑或地雷, 其等级和对应的科技树中项目一致.
    • 解锁原型防御建筑只能通过三选一项目.
    • 选定之后, 您仍可以花费钻石来更改选定的项目.
    • 更改之后, 新选定的项目将替换之前的项目.
    • 每次更改选择所需的钻石花费都会比前一次高(有上限), 同时所有三选一项目都会同步该花费.

    匹配

    战斗母舰是根据您的星阶及荣誉之星数为您匹配对手.

    • 系统会首先尝试为您匹配星阶接近的对手.
    • 系统会尽量为您匹配引擎室数量相同的对手.
    • 在某些情况下, 系统也会放宽匹配机制, 以保证为您更快匹配到对手.

    准备战斗

    • 在出海对战之前, 您可以在”出战部队”界面进行战斗准备. 在这个界面中, 您可以选择参加对战的部队, 英雄.
    • 先点击登陆艇, 然后点击您希望装载的部队, 就可以为您的登陆艇装载部队了.
    • 每个部队都有对应规模的储备规模, 您可以在科技树中升级该部队以扩大该部队的储备规模.
    • 如果您的部队数量不足, 您依然可以选择吧剩余的部队装载上登陆艇, 只是不能装满.
    • 当您在科技树中对某一部队进行了升级, 或是部分士兵在对战中阵亡, 新的部队将自动训练并补充至您的储备部队中.

    可以加速训练

    • 您可以花费钻石立即填满某一部队的储备部队.
    • 如果您启用了瞬时训练, 您的储备部队将保持在满员状态!

    注意

    • 在科技树中升级英雄时, 英雄技能将自动解锁和升级.
    • 侦查敌人的时候, 只要您还没有开始进攻, 就还可以更换出战部队.
    • 但是, 此时您无法花费钻石来补充储备规模.

    战斗母舰宝箱

    • 战斗母舰宝箱可以开出主基地资源, 情报, 原型模块, 能量水晶颗粒, 能量水晶碎片, 钻石和英雄券等, 以及能帮您强化战斗母舰的解锁币和升级币.
    • 摧毁一定数量的引擎室, 您就可以领取一个战斗母舰宝箱.
    • 您一次性最多可以存储4个战斗母舰宝箱. 领取完一个战斗母舰宝箱后, 您需要等待一定时间才会有新宝箱送达.

    特遣队

    简介

    • 当司令部达到6级后, 您可以组建自己的特遣队, 或加入其它特遣队.
    • 特遣队有四种规模: 5人, 10人, 25人, 50人.
    • 创建特遣队之后您可以扩大特遣队规模, 但不可缩减.
    • 同一规模的特遣队处于同一排行榜.

    管理

    • 指挥官: 特遣队的创建者会被默认为指挥官, 指挥官可以将队员晋升为军官, 副指挥官, 甚至指挥官(自己会自动降职为副指挥官); 可以踢出成员, 军官和副指挥官.
    • 副指挥官: 几乎拥有指挥官的所有权限, 但不能将指挥官及其他副指挥官降职或踢出特遣队, 也不能晋升任何人为指挥官.
    • 军官: 可以踢出成员和接受入队申请; 军官每隔20分钟可以踢一次成员.
    • 如果指挥官在没有任命新指挥官的情况下离开了特遣队, 加入特遣队时间最久的副指挥官将被自动晋升为指挥官; 如果特遣队没有副指挥官, 那么加入时间最久的军官将被自动晋升为指挥官, 以此类推…
    • 当一个成员被踢出特遣队时, 该成员需要等待24小时才能提交新的加入特遣队申请. 但指挥官, 副指挥官和军官可随时使用邀请归队的功能邀请被踢出的玩家立即归队.
    • 如果您在特遣队任务过程中加入特遣队的话, 您将无法参与此次任务, 只能等待下次特遣队任务.

    任务和情报

    • 只有指挥官, 副指挥官和军官才能发起特遣队任务.
    • 必须收集足够的情报, 才能发起特遣队任务.
    • 发起特遣队任务的情报数量要求取决于特遣队的规模(例如开启”定点运输”任务5人队需要10张情报, 10人特遣队需要15张情报)
    • 情报可以通过攻打地图上的岛屿或者主基地防御成功获得, 也可以通过补给箱, 每日奖励及每日时间等获得.
    • 即使您离开特遣队, 特遣队扔可以保留您收集的情报.

    关于摧毁能量中心

    • 在执行特遣队任务时, 您可以在能量基地中看到很多的建筑. 每摧毁一座建筑, 将获得1个战队积分.
    • 您的首要目标是能量中心, 摧毁它能引发连锁反应, 从而摧毁整个能量基地, 并获得所有可得的战队积分.
    • 指挥官, 副指挥官和军官可以使用情报随机破坏一定的建筑物.
    • 进入观战模式, 可实时观看其他队员发起的攻击.

    奖励

    • 在维持23小时55分钟的特遣队任务结束后, 您将收到运输船给您运送的奖励. 您可以将您的特遣队奖励保留至下一次特遣队任务即将结束之前, 但任务结束后, 您之前的尚未领取的奖励将会被新的奖励覆盖.
    • 如果您在执行特遣队任务期间离开了特遣队, 即使您在任务结束之前重新归队, 您扔将无法获得战队奖励.

    树木和石头(障碍物)

    • 障碍物是您基地的石头和树木, 可以用金币移除石头或木头.
    • 资源岛中的障碍物也可以由您(当您控制资源岛时)或对手(当Ta控制资源岛时)清除
    • 只有在达到最低要求司令部等级后才能清除某些障碍
    • 清除时, 不同类型的障碍物会产生不同数量的资源.
    • 清除障碍所需的金数量越大, 每金币获得的资源越少. 基本上, 最便宜的障碍物会带给您最大的性价比
    • 障碍物不会在资源岛上重生, 但是树木会在您的基地上随机重生
    • 每隔13个小时, 有40％的机会会在您的基地上生成一棵新树
    • 只有当前可以清除的树木类型才有机会重生. 例如: 如果您的总部位于12级或以下, 红树林不会重新生成.
    • 您的基地最多可以拥有60棵树
