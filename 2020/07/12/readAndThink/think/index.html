    思考

    绝望沙洲

    在我扩展知识领域中遇到不懂的地方, 我会把需要查询的关键字丢给搜索引擎, 它会返回给我更多不懂的东西, 很难查询到相关的线索!
    最糟糕的是我不知道我什么不知道(I don’t know what i don’t know).
    于是 学也学不完, 越学越不懂 的无力感排山倒海而来. 就像进入了绝望沙洲.
    只要在绝望沙洲中熬过去, 就能够累积足够的经验, 达到一定的水准, 就能准确判断问题的方向切入核心, 工作效率因而有所增进, 知识广度也会慢慢聚焦.

    也许我已经学会该学的, 做了该做的, 成为一个名副其实的开发工程师, 却总觉得自己资质平庸根基不稳, 虽然有成功打造出一个个专案而信心回增, 但老是感叹自己与心目中的专业工程师仍有一大段距离.

    I learned the value of hard work by working hard.

    你家的灯亮着么

    人们很少知道自己需要什么, 直到 ……你给了他们需要的东西…

    • 如果在你对问题的理解中,你想不出至少三种可能出错的东西, 那么你并没有真正的理解这个问题.
    • 如果某人能够解决这个问题, 但是他本人却并不会遇到这一问题时, 那么你首先要做的就是让他也感受到这一问题.
    • 我们永远没有足够的时间来把它做好. 但是我们有足够的时间来重新做一遍., 但是, 因为我们不是总有机会去重来一遍, 我们必须做的更好一些, 换言之: 我们永远没有足够的时间来考虑我们是否真的需要它, 但是我们总是有足够的时间为之后悔..
    • 鱼总是最后一个看到水的…..
    • 勤于思考和积极工作都需要时间
    • 三十六计更多的时候是被当成方法论来读的, 其根源在于’计谋’, 本身只是方法, 而不是战略.

    第二象限

    想一想，当前重要紧急的事情是怎么出现的？

    是不是自己提前做好铺衬之后，这些紧急的事情，可能就不会出现。

    学习

    • 智慧是任何人都抢不走的，你只要活着，智慧就永远跟着你。
    • 休息日要达到休息的目的，松弛紧张的神经和肌肉，净化浮躁的心灵，以更好地恢复良好的工作状态。
      如果休息日里没有将自己的状态调整好，那恐怕就很难实现真正意义上心灵与身体的改善。
    • 一般的学习只是一种模仿，而没有任何的创新，学习应该以思考为基础。思考是由怀疑和答案所组成的;学习便是经常怀疑，随时发问。怀疑是智慧的大门，知道得越多，就会怀疑得越多，而问题也就随之增加，所以发问使人进步

    达成目标

    1. 希望达到的行为目标(是行为目标)
    2. 现在正在做的, 与目标相反的行为
    3. 认为做这些与目标相反的行为, 有什么潜在的好处, 或者换一种问法, 如果不做这些事情, 你担心的最糟糕的事情是什么?
    4. 内心里有什么重大假设
