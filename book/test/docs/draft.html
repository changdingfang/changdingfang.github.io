数组和指针的区别

  • 数组和指针是两种不同的类型
  • 指针表示某种数据类型的数据存储的内存地址, 数组表示若干个相同数据类型的元素在连续内存中存储的一种形态

  • 使用 sizeof 计算结果不同 

      char arr[10]
  char *p = arr;

  sizeof arr == 10
  sizeof p == 8
