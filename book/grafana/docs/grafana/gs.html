数据源(Data Source)

数据源是为Grafana展示数据提供的数据仓库

Grafana支持的数据源

  • AWS CloudWatch
  • Azure Monitor
  • Elasticsearch
  • Google Cloud Monitoring
  • Graphite
  • InfluxDB
  • Loki
  • Microsoft SQL Server (MSSQL)
  • MySQL
  • OpenTSDB
  • PostgreSQL
  • Prometheus
  • Jaeger
  • Zipkin
  • Tempo
  • Testdata

仪表板(Dashboard)

仪表板由一个或多个面板组合而成, 形成一个大的仪表板, 用于展示各种信息

仪表板里面的面板可以随意的拖动, 缩放和放大, 可操作性很高.
