    代码: 

    <font face="黑体">我是黑体字</font>  
<font face="微软雅黑">我是微软雅黑</font>  
<font face="STCAIYUN">我是华文彩云</font>  
<font color=red>我是红色</font>  
<font color=#008000>我是绿色</font>  
<font color=Blue>我是蓝色</font>  
<font size=5>我是尺寸</font>  
<font face="黑体" color=green size=5>我是黑体，绿色，尺寸为5</font>

    我是华文彩云

    我是绿色
    我是蓝色
    我是尺寸

    超链接

    行内式

    []里写链接文字, ()里写链接地址, ()中的 “” 中可以为链接指定title属性, title属性可加可不加. title属性的效果是鼠标悬停在链接上会出现指定的 title文字. “链接文字“这样的形式. 链接地址与链接标题前有一个空格

    代码: 

    
查看[linux常用命令行](/2020/07/12/notes/linux/linux_cmd_line) 
查看[linux常用命令行](/2020/07/12/notes/linux/linux_cmd_line)

    效果:

    查看linux常用命令行
    查看linux常用命令行

    参考式

    参考式链接分为两部分, 文中的写法 [链接文字][链接标记], 在文本的任意位置添加[链接标记]:链接地址 “链接标题”, 链接地址与链接标题前有一个空格.

    如果链接文字本身可以做为链接标记, 你也可以写成[链接文字]

    规则:
    [链接文字]: 链接地址

    代码: 

    我喜欢看[linux命令行]  
我喜欢逛[百度贴吧]  

[linux命令行]: /2020/07/12/notes/linux/linux_cmd_line "linux命令行"
[百度贴吧]: https://tieba.baidu.com/ "百度贴吧"

    效果:

    我喜欢看linux命令行
    我喜欢逛百度贴吧

    自动链接

    用 <> 包起来, 就会自动转成链接.

    代码:

    效果:

    https://www.baidu.com

> 如果中间有空格或者特殊符号，改为“-”连接

    定义型列表

    定义型列表由名词和解释组成. 一行写上定义, 紧跟一行写上解释. 解释的写法: 紧跟一个缩进(Tab或空格)

    代码: 

    常丁方
: 男, 1999年1月16日出生(农历一九九八年十一月二十九), 戊寅年, 乙丑月, 戊辰日, 兔年

代码块
: 代码  

        printf("hello world"); // 左边有两个tab键(或8个空格)

    效果:

    常丁方
    男, 1999年1月16日出生(农历一九九八年十一月二十九), 戊寅年, 乙丑月, 戊辰日, 兔年

    代码 
        printf("hello world"); // 左边有两个tab键(或8个空格)

    注脚

    在需要添加注脚的文字后加上脚注名字[^注脚名字], 称为加注。. 然后在文本的任意位置(一般在最后)添加脚注, 脚注前必须有对应的脚注名字.
    即使你没有把注脚写在文末, 经Markdown转换后, 也会自动归类到文章的最后.

    代码: 

    Markdown[^1]可以效率的书写文档, 直接转换成HTML[^2], 你可以使用有道云笔记[^3]进行书写.


[^1]: Markdown是一种纯文本标记语言
[^2]: HyperText Markup Language 超文本标记语言
[^3]: 有道云笔记(原有道笔记)是2011年6月28日网易旗下的有道推出的个人与团队的线上资料库. 有道云笔记采用了增量式同步技术, 即每次只同步修改的内容而不是整个笔记

    效果:

    Markdown^1可以效率的书写文档, 直接转换成HTML[^2], 你可以使用有道云笔记[^3]进行书写.
