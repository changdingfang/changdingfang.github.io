
# shuf

产生随机的排列

## 语法

```shell
shuf [选项] [参数]
```

## 选项

- -e    
    将每个ARG视为输入行   
- -i    
    将数字范围LO(最低)到HI(最高)之间的作为输入行   
- -n    
    只输出当前COUNT行   
- -o   
    将结果写入到文件而不是标准输出   
- -r   
    输出行可以重复   
- -z    
    行终止符为NULL(空字符)而不是默认的换行符   

## 参数

- 文件名  
    可以是多个文件名，用空格隔开
- ARG   
    作为输入行的字符串，可以为任意数量


## 使用示例

产生1-10之间的数字，输出5个

```shell
shuf -i 1-10 -n 5
```
