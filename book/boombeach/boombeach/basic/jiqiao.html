扎卡定理

当妹子的射程攻击范围有目标的时候，妹子不是无脑地从近到远攻击目标，而是会动脑子的。妹子选择目标的优先权如下

  1. 防御建筑 > 升级中的防御建筑 > 无害建筑
    此条无法判断优先次序的进入下一条
  2. 残血(百分比)少的建筑 > 残血多的建筑 > 无损建筑(=残血量100%)
    此条无法判断优先次序的进入下一条
  3. 距离近的 > 距离远的

获取资源计算

注: 这个可能不准确或者根本就不对

a = 显示的总资源
v = 保险库存储的资源
p = 生产建筑存储的资源
x = 保险库保护百分比
r = 系统奖励资源
r2 = 减少系数
s = 雕像加成百分比
t = 部落加成百分比

可获得资源 = (( a - v + p ) (1 - x) + r ) ( 1 - r2 ) * ( 1 + s + t )

秒掉时间钻石消耗计算公式

公式. 1.25 * 20.147 * (小时)0.7513 ↩

总钻石消耗 = 25.18375 * (总时间小时)0.7513

当前钻石消耗 = (剩余时间 / 总时间) * 总钻石消耗
