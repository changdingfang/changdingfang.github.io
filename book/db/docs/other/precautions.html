注意注意

  • 空值

  • 参考

空值

select * from tb where name != 'dingfang'

这个查出的数据不仅不包含 dingfang，而且也不包含空值数据。

应该改为

select * from tb where (name != 'dingfang' or name is null)

时间

select * from tb where t <= '2021-03-29'

这个时间一般意味着是 2021-03-29 00:00:00，所以这里用 <= 会包含29日一个整点的时间数据，看起来会很奇怪，有些场景这样会认为是错误的。

所以取时间一般是左闭右开区间。 

select * from tb where  t >= '2021-03-29' and t < '2021-03-30'
