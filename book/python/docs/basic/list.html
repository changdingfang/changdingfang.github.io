list完全相等比对

内容一样, 顺序不一样

假设有如下两个列表 

list1 = ["one", "two", "three"]  
list2 = ["two", "three", "one"]  
`

不改变本身: 可以使用 sorted 方法排序在进行比对 

sorted(list1) == sorted(list2)

改变本身: 使用list自带的sort排序后比对。

list1.sort() == list2.sort()

list转字符串

# 1. list中全部为字符串
li1 = ['hello', 'world', 'nihao']
s1 = ",".join(li1) # 转换成以逗号分割的字符串

# 2. list中有数字
li2 = [1, 2, 3, 4]
s2 = [str(x) for x in li2]
