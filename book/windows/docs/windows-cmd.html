PowerShell

在PowerShell可以运行cmd的命令，但可能会出现冲突(由于命令名称和优先级可能会优先运行powershell的命令)，这时可以使用命令的扩展名来执行，指定使用哪个命令。

示例 

# 这条命令在cmd中运行没有什么问题， 但是在powershell中会报错，因为powershell中会优先使用Set-Content命令
sc query lanmanserver

# 解决 使用扩展名
sc.exe query lanmanserver

# 或者指定使用cmd来执行命令
cmd /c "sc query lanmanserver"
