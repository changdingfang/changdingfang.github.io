constexpr

该关键字在c++11已经出现，但在c++11里面constexpr函数有且仅有一条return语句，现在可以写多条语句了。 

/* 这在cpp14里面是合法的 */
constexpr int foo(const int n)
{
    if (n == 1 || n == 2)
    {
        return 1;
    }
    int tmp = 4;
    tmp++;
    return foo(n - 1) + foo(n - 2);
}

int main(void)
{
    int k = foo(8);

    return 0;
}
